# 게시판 스킨

게시판 스킨은 게시판의 목록, 글 내용 보기, 댓글 및 댓글 쓰기, 글 쓰기 폼의 스킨으로 구성되어 있다.

```shell
skin
└── board
    └── basic # 'basic'이라는 이름을 가진 스킨 폴더
        ├── list.skin.php # 글 목록
        ├── view.skin.php # 글 내용 보기
        ├── view_comment.skin.php # 댓글 및 댓글 쓰기 폼
        └── write.skin.php # 글 쓰기
```
